Desktop

Dark mode
- Black page header
- Dark gray background
- Lighter gray buttons and table headers
- Rows are dark gray (background) - hover on lighter gray

Light mode
- Light gray page header
- White background
- Same light gray for table headers
- Rows are white (background) - hover on same light gray


Mobile

Dark mode
- background black, same black as the app navigation in the desktop version
- rows/cards are the same dark gray that we used for the background in desktop version
- hovers/buttons are the lighter gray that we used for surfaces in the desktop version
- top and bottom nav bars are black (like app nav) but then they change to the dark gray we used as background with a bit of transparency and blur if there is something behind them

Light mode
- background light gray, the same we used as app navigation
- rows/cards are the same white that we used for the background in desktop
- hovers/buttons are the gray we used for surfaces in desktop
- top and bottom nav bars are light gray (like app nav) but they change to the light gray we used as background with a bit of transparency and blur if there is something behind them
